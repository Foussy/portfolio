## Project 1 : pyapod
### python3 | PyQt
This little software is a cross platform front-end of the ![Atronomy Picture of the day website](https://apod.nasa.gov/apod) posting photos & videos Space related everyday since June 1995. It is designed with python3 and PyQt5 library and it uses the API developped by NASA.
